# BBS项目介绍
这个是一款小巧的博客写作系统，编辑器为markdown editor，博客主题移植了Hexo Next的样式，简洁大方。

主要特性：

- 编辑文章支持实时预览
- 为新博文自主添加分类和标签
- 支持本地上传图片文件
- 博文时间线视图
- 博文分类视图和标签视图
- 支持全文检索
- 支持分页查看
